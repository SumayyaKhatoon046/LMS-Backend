# 🚀 SkillForge AI

### 🤖 An AI-Powered Learning Management System

SkillForge AI is a modern **Learning Management System (LMS) Backend** built with **Node.js, Express.js, MongoDB, Mongoose, JWT Authentication, and Swagger API Documentation**.

The platform enables secure authentication, role-based access control, course management, lecture management, quizzes, student progress tracking, certificate generation, and teacher dashboards. It is designed with scalability in mind and will be enhanced with AI-powered learning features in future versions.

---

# 📖 About

SkillForge AI is a backend application for an AI-powered Learning Management System (LMS). It allows educational institutions and online learning platforms to manage courses, lectures, quizzes, student progress, certificates, and teacher dashboards efficiently.

The project follows a secure **Role-Based Access Control (RBAC)** architecture using **JWT Authentication** and is built using modern backend technologies. Future versions will integrate Artificial Intelligence to provide personalized learning experiences and intelligent educational tools.

# ⚙️ Installation

### 1️⃣ Clone Repository

```bash
git clone https://github.com/SumayyaKhatoon046/LMS-Backend.git
```

### 2️⃣ Move into Project

```bash
cd LMS-Backend
```

### 3️⃣ Install Dependencies

```bash
npm install
```

### 4️⃣ Create Environment Variables

Create a `.env` file in the project root.

```env
PORT=5000
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_secret_key
```

### 5️⃣ Start Server

```bash
npm start
```

Server will start at:

```text
http://localhost:5000
```
