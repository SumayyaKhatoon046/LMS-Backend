# 🚀 SkillForge AI

### An AI-Powered Learning Management System

SkillForge AI is a modern Learning Management System (LMS) backend built with Node.js, Express.js, MongoDB, and JWT Authentication. It provides secure authentication, course management, lecture management, quizzes, progress tracking, certificate generation, and teacher/student dashboards. The project is designed to be scalable and will be enhanced with AI-powered learning features in future versions.
